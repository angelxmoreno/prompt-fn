Design Colors
=============

The Docusaurus theme is locked to the same palette used across the main repo. Use these values when updating the logo, social images, or any additional illustrations so everything stays cohesive.

| Token | Hex | Usage |
| --- | --- | --- |
| `--ifm-color-primary` | `#0f172a` | Base navy, hero background, feature cards |
| `--ifm-color-primary-light` | `#131c31` | Gradient stop, hovered elements |
| `--ifm-color-primary-lighter` | `#18223b` | Highlight ring / borders |
| `--ifm-color-primary-dark` | `#0c1323` | Shadow tone |
| `--ifm-color-primary-darkest` | `#050910` | Deep accents / outlines |
| `--pf-hero-gradient` light stop | `#2563eb` | Hero gradient, CTAs |
| Secondary CTA | `#1d4ed8` | Filled buttons, outlines |
| Light text | `#f8fafc` | Hero headline, feature card copy |
| Muted text | `#e2e8f0` | Hero subtitle |
| Page background | `#f8fafc` | Light-mode body |
| Dark page background | `#020617` | Dark mode body/footer |
| Card highlight (light) | `#bfdbfe` | Feature label accents |
| Card highlight (dark) | `#93c5fd` | Feature label accents (dark mode) |
| Ghost link | `#1e3a8a` | Hero “Star on GitHub” link |
| Dark ghost link | `#bfdbfe` | Ghost link in dark mode |

When creating new artwork, keep at least one dark navy (#0f172a → #050910) and one electric blue (#1d4ed8 / #2563eb) so the assets blend with the UI. Use `#f8fafc` for negative space or line work to stay readable on the gradient backgrounds.\*\*\*
